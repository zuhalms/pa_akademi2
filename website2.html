Tentang SMART-BA

SMART-BA adalah sistem informasi pembimbingan akademik berbasis web yang dirancang untuk memfasilitasi proses bimbingan akademik antara mahasiswa, dosen pembimbing akademik (PA). Sistem ini menyediakan platform terintegrasi untuk mengelola konsultasi, monitoring akademik, dan dokumentasi kegiatan pembimbingan.

URL Repository: https://github.com/zuhalms/pa_akademi

Created by : Zuhal Samas

‚öôÔ∏è Panduan Instalasi

Prerequisites

Server Requirements:
‚Ä¢ PHP 7.4 atau lebih tinggi
‚Ä¢ MySQL 5.7 atau MariaDB 10.2+
‚Ä¢ Apache / Nginx Web Server
‚Ä¢ PHP Extensions: mysqli, gd, mbstring, session
‚Ä¢ Minimum RAM: 512MB
‚Ä¢ Disk Space: 50MB minimum

Langkah Instalasi

1
Clone Repository
git clone https://github.com/Zuhal Samas/pa_akademi.git
atau download file pa_akademi.zip (4.6MB)
2
Extract Files
Extract ke direktori web server (htdocs/xamp/public_php)
3
Create Database
Buat database MySQL baru:
CREATE DATABASE pa_akademi;
4
Import Database
Import file db_pa_akademi.sql ke database:
mysql -u root -p pa_akademi < db_pa_akademi.sql
atau gunakan phpMyAdmin untuk import
5
Configure Database
Edit file config.php dan db_connect.php:
- Set database host
- Set database username
- Set database password
- Set database name
6
Set Permissions
Berikan write permission untuk folder:
- dokumen/ (chmod 755)
- assets/uploads/ (chmod 755)
7
Access Application
Buka browser dan akses:
http://localhost/pa_akademi/
atau sesuai dengan domain Anda
8
Default Login
Gunakan kredensial default dari database:
Admin: Lihat tabel users
Dosen: Lihat tabel dosen
Mahasiswa: Lihat tabel mahasiswa
